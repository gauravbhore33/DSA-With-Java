1) Lock your daily workflow

After every lecture:

git add .
git commit -m "Day 6: <topic you learned>"
git push

Keep commits meaningful. One lecture = one clean commit.

2) Make your repo look intentional (README upgrade)

Open README.md and turn it into a progress tracker:

- Taking user input
